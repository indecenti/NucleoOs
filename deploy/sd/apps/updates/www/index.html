Firmware update (over the air)

Pick a nucleoos.bin and install it wirelessly — no USB cable. The image is written to the spare flash slot and only booted if it is valid; if the new firmware fails to start, the device automatically rolls back to this one. So it always stays bootable.

⚠ Keep the device powered during the update. It reboots automatically when done.

App & shell updates

The shell and apps live on the SD card, so they update independently of the firmware — either by syncing the card from a PC (deploy.ps1 -To <sd>) or over the network with npm run push-ota -- --host http://<ip>. After a web update, just reload the shell.
