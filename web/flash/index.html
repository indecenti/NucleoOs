1-click firmware install

Chrome or Edge on a desktop, Cardputer plugged in via USB-C.

Your browser can't flash over USB. Use Chrome or Edge on a desktop computer (Web Serial isn't available on Firefox, Safari, or phones). This page needs to be served over HTTPS to access the serial port.
🔌 USB-C cable 💾 microSD (FAT32) 🌐 Chrome / Edge desktop

Full setup — 3 steps

  1. Flash the firmware

    Click Connect & install NucleoOS above, pick the Cardputer's serial port, and confirm. It writes the latest firmware for you.

    When it finishes, press the Cardputer's reset button (or unplug/replug USB) to start it. You can't brick it — the previous firmware stays safe and you can always re-flash.

  2. Prepare the microSD card

    NucleoOS serves all its apps and data from the card. Format a microSD as FAT32, then extract the SD payload to its root — you should end up with /apps /www /system /data. Insert it before powering on.

    ⬇ Download the SD payload (.zip)

  3. First boot

    The Cardputer guides you: choose a language, join your Wi-Fi (typed on the keyboard), set a name. Then the screen shows an IP address — open it in a browser on the same Wi-Fi for the full desktop console.

    Skipped Wi-Fi? The Cardputer starts its own network NucleoOS-xxxx with a password shown on screen — connect a phone or PC to it and open the address displayed.

Heads up: the -sd.zip is the SD-card payload, not firmware — it goes on the microSD, never into a flasher. This installer flashes only the firmware image.
